# Data

Raw data capsules downloaded from each round, one subfolder per round.

CSV / JSON / Parquet files here are gitignored (see `.gitignore`) — drop them
locally after downloading from the Prosperity platform.

Expected Round 1 filenames (consumed by `backtesting/round1_backtest.py`):

```
data/round1/prices_round_1_day_-2.csv
data/round1/prices_round_1_day_-1.csv
data/round1/prices_round_1_day_0.csv
```
